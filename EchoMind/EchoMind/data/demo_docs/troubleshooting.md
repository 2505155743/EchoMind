# EchoMind 故障排查手册

## 应用崩溃

如果应用频繁崩溃，请按以下步骤排查：

1. 检查设备存储空间是否充足（至少需要 500MB 可用空间）
2. 清除应用缓存：设置 → 应用管理 → EchoMind → 清除缓存
3. 确认应用版本为最新（当前最新版本 v2.0.3）
4. 如果以上步骤无效，请卸载后重新安装

## 登录问题

### 401 认证失败
- 确认用户名和密码正确
- 检查是否开启了大写锁定
- 尝试使用「忘记密码」功能重置
- 如果使用第三方登录，确认授权未过期

### 403 权限不足
- 确认账户未被冻结
- 检查是否在允许的 IP 范围内
- 联系管理员确认权限配置

## 支付问题

### 支付失败
- 确认银行卡余额充足
- 检查是否开启了网上支付功能
- 尝试更换支付方式
- 如果使用信用卡，确认未超过单笔限额

### 重复扣款
- 不要重复提交订单
- 如果确认被重复扣款，系统会在 24 小时内自动退回多余款项
- 超过 24 小时未退回，请联系客服提供订单号处理

## 网络问题

### 页面加载慢
- 检查网络连接是否正常
- 尝试切换 WiFi 和移动数据
- 清除浏览器缓存
- 尝试使用其他浏览器

### 连接超时
- 检查是否使用了 VPN（部分 VPN 可能影响连接）
- 尝试重启路由器
- 如果是公司网络，确认防火墙未拦截
